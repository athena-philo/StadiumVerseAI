# ⚽ StadiumVerseAI

> **An AI-powered Smart Stadium Platform for enhancing fan experience and intelligent stadium management.**

StadiumVerseAI is a full-stack AI application developed as part of the **Hack2Skill Build with AI Challenge**. The platform combines Generative AI with modern web technologies to deliver intelligent assistance, real-time crowd insights, multilingual accessibility, and emergency support for large-scale sporting events.

---

## 🚀 Features

### 🤖 AI Stadium Assistant
- Intelligent chatbot powered by Google Gemini
- Answers stadium-related queries
- Assists users with venue information and navigation

### 🗺️ AI Match Planner
- Generates personalized match-day plans
- Helps fans organize their stadium visit efficiently

### 👥 Crowd Intelligence Dashboard
- Displays crowd density
- Shows queue lengths
- Monitors seating occupancy
- Highlights congested stadium zones

### 🚨 Emergency Response Assistant
- Generates AI-assisted emergency response plans
- Supports multiple emergency scenarios

### 🌍 Multilingual Support
- Translates stadium announcements into multiple languages

### 📢 Live Stadium Announcements
- Displays important announcements in real time
